# CaesarSharp

Приложение для шифрования и взлома шифра Цезаря.

## Возможности

- Шифрование и дешифрование текста
- Автоматический взлом шифра по частотному анализу
- Поддержка 13 языков: русский, английский, немецкий, французский, испанский и др.
- Графический интерфейс (GUI) и консольный интерфейс (CLI)
- Пакетная обработка файлов

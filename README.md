Мощное и гибкое приложение на языке C# для шифрования, дешифрования и криптоанализа (взлома) классического шифра Цезаря. Проект сочетает в себе высокую производительность ядра, кроссплатформенную логику и современные интерфейсы управления.

---

## 🔥 Ключевые возможности

* **Двустороннее шифрование:** Быстрое шифрование и дешифрование текстов любого объема с заданным шагом (ключом).
* **Smart Cracking (Частотный анализ):** Автоматический взлом зашифрованного текста без знания ключа на основе частотных характеристик букв конкретного языка.
* **Пакетная обработка файлов:** Возможность массового шифрования или расшифровки текстовых документов в один клик или через одну консольную команду.
* **Dual Interface (GUI / CLI):** 
  * Современный графический интерфейс для десктопа.
  * Быстрый и удобный консольный интерфейс для автоматизации и продвинутых пользователей.

## 🏗️ Структура проекта

Архитектура приложения разделена на независимые слои для обеспечения модульности и слабой связанности:

* 📦 `CaesarSharp.Core` — изолированная библиотека классов, содержащая всю математическую и криптографическую логику, алгоритмы частотного анализа и словари для языков.
* 🖥️ `CaesarSharp` — исполняемый проект, объединяющий под одной оболочкой графический интерфейс (построенный на **WinUI 3**) и интерфейс командной строки (**CLI**).

---

## 🛠️ Стек технологий

* **Язык разработки:** C# (.NET 8 / .NET 9)
* **Графическая подсистема:** WinUI 3 (Windows App SDK)
* **Пакетная обработка:** System.IO (асинхронный поток данных)
